# 排班日历 - 上6休2

一个简洁的排班日历 Web 应用，支持「上6休2」循环排班模式，可在日历上直观查看每天的班次。

## 功能特点

- 📅 **日历视图**：直观显示整月排班情况
- 🎨 **颜色区分**：休息（绿）、上班（橙）、夜班（蓝）一目了然
- 🔗 **分享功能**：生成含排班信息的链接，同事打开即看
- 📊 **今日信息**：显示今天班次、下次休息、下次夜班
- 📈 **月度统计**：统计本月上班/休息天数
- 💾 **本地保存**：起始日期自动保存在浏览器本地

## 排班规则

8天一循环：
- **第1天**：下午班 + 夜班
- **第2~5天**：正常上班
- **第6天**：早班（下午下班）
- **第7~8天**：休息

## 使用方法

1. 打开应用，设置「排班起始日」（第1天：下午+夜班）
2. 日历自动按「上6休2」循环标记所有日期
3. 点击「分享」按钮，复制链接发给同事

## 部署

本应用为纯静态 HTML，可部署到任何静态托管服务：

- **EdgeOne Pages**：绑定 GitHub 仓库自动部署
- **GitHub Pages**：启用 GitHub Pages 即可
- **Vercel / Netlify**：拖拽上传或连接仓库

## 本地运行

直接用浏览器打开 `index.html` 即可，无需服务器。
